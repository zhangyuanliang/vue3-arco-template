# vue3-admin-template

vue3-admin-template

## 安装使用

- 获取项目代码

```bash
git clone http://git.dbhs.com.cn/zhangyuanliang/vue3-admin-template.git
```

- NodeJs

```bash
版本 >=14.0.0
```

- 安装依赖

```bash
cd vue3-admin-template

pnpm install

```

- 运行

```bash
pnpm run start
```

- 打包

```bash
pnpm run build
```
